# SAM3-LoRA 项目总结

## AI研究组 - 蒙固国王科技大学（KMUTT）

***

## 📦 项目概述

**SAM3-LoRA** 是一个使用低秩适配（LoRA）技术对Meta的Segment Anything Model 3进行高效微调的完整框架。该实现使KMUTT的研究人员和开发者能够将848M参数的SAM3模型适配到自定义分割任务，同时只训练不到1%的参数。

## ✨ 关键特性

### 1. 灵活的LoRA配置

- **选择性组件应用**：将LoRA应用于任意组合：
  - 视觉编码器（32层ViT）
  - 文本编码器（概念提示）
  - 几何编码器（边界框）
  - DETR编码器（视觉-文本融合）
  - DETR解码器（对象查询）
  - 掩码解码器（像素级掩码）

### 2. 配置管理

- **基于YAML的配置**：易于阅读和修改
- **CLI覆盖**：从命令行覆盖任何参数
- **三种预配置模板**：最小、平衡、完整

### 3. 训练基础设施

- **混合精度训练**：FP16/BF16支持加快训练
- **梯度累积**：处理更大的有效批量大小
- **自动检查点**：基于验证IoU保存最佳模型
- **学习率调度**：余弦、线性或恒定调度器

### 4. 数据支持

- **COCO格式转换器**：将COCO注释转换为SAM3格式
- **YOLO格式转换器**：将YOLO注释转换为SAM3格式
- **数据集验证**：训练前验证数据集完整性
- **自定义数据集支持**：易于扩展到其他格式

### 5. 推理与部署

- **简单推理API**：易于使用的推理脚本
- **多种提示类型**：文本提示、边界框或两者结合
- **批处理**：高效处理多张图像
- **轻量级检查点**：仅保存LoRA权重（10-50MB）

## 🎯 使用场景

### 快速开始（5个命令）

```bash
# 1. 设置
./quickstart.sh

# 2. 准备数据
python prepare_data.py create --output_dir data

# 3. 训练
python train_sam3_lora.py --config configs/base_config.yaml

# 4. 推理
python inference.py \
  --lora_weights outputs/sam3_lora/best_model/lora_weights.pt \
  --image test.jpg \
  --text_prompt "yellow school bus"

# 5. 完成！
```

### 训练配置

| 配置     | 命令                                          | 参数           | 训练时间    |
| ------ | ------------------------------------------- | ------------ | ------- |
| **最小** | `--config configs/minimal_lora_config.yaml` | 500K (0.06%) | 约1小时    |
| **平衡** | `--config configs/base_config.yaml`         | 4M (0.47%)   | 约3-4小时  |
| **完整** | `--config configs/full_lora_config.yaml`    | 15M (1.77%)  | 约8-10小时 |

*基于1K训练图像、10个周期、RTX 3090*

## 🛠️ 技术实现

### LoRA算法

```python
# 原始前向传播
output = W @ x

# LoRA前向传播
output = W @ x + (B @ A @ x) * (alpha / rank)
#         ↑       ↑           ↑
#      冻结   可训练    缩放
```

### 内存优化技术

- **混合精度训练**：使用BF16/FP16减少50%内存
- **梯度累积**：用更少内存实现更大有效批量
- **选择性冻结**：只训练LoRA参数，冻结主干
- **激活检查点**：权衡计算换内存

### 兼容性

- ✅ 与SAM3官方模型完全兼容
- ✅ 支持SAM3的所有分割功能
- ✅ 可与SAM3研究社区集成
- ✅ 支持导出到ONNX（待验证）
